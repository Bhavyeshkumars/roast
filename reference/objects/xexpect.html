expect(expected_failures=None, expected=None, wait_for_prompt=True, err_index=0, timeout=200, err_msg=None)[source]¶

Seeks through the stream until a pattern is matched.

Parameters

  • expected_failures (Union[None, List[str], str]) – list of failure patterns or a single fail pattern string to be expected. Defaults to None.

  • expected (Optional[List[str]]) – List of patterns or a single pattern string to be expexted. Defaults to None.

  • wait_for_prompt (bool) – Specify if prompt has to be expected. Defaults to True.

  • err_index (int) – Length of expected_failures list. Defaults to 0.

  • timeout (int) – Waits for mentioned timeout for the expected string. Defaults to 200.

  • err_msg (Optional[str]) – Custom error message. Defaults to None.

Return type

int

runcmd(cmd, expected_failures=None, expected=None, wait_for_prompt=True, timeout=200, err_msg=None, retries=1)[source]¶

Sends command on to the console with expected failures and expected output and returns the index of expected string.

Parameters

  • cmd (str) – Command to be executed on the console

  • expected_failures (Union[None, List[str], str]) – List of failure patterns or a single fail pattern string to be expected. Defaults to None.

  • expected (Optional[List[str]]) – List of strings or string. Defaults to None.

  • wait_for_prompt (bool) – To specify if prompt has to be expected after expected pattern. Defaults to True.

  • timeout (int) – Waits for mentioned timeout for the expected string. Defaults to 200.

  • err_msg (Optional[str]) – Error message. Defaults to None.

  • retries (int) – Number of retries. Defaults to 1.

Return type

int

Returns

Index of the expected string.

runcmd_list(cmd_list, expected=None, timeout=200, err_msg=None)[source]¶

Sends list of commands to the console and expects the specified string.

Parameters

  • cmd_list (List[str]) – List of commands to be run on console.

  • expected (Optional[List[str]]) – Expects for the same string for all the commands. If nothing is specified it expects for prompt. Defaults to None.

  • timeout (int) – Waits for mentioned timeout for the expected string. Defaults to 200.

  • err_msg (Optional[str]) – Optional custom error message. Defaults to None.

Return type

None
