roast.utils.roast_utils.filter_keys(keys)[source]¶

This api is used to take a string or a list of one string and split it appropriately to get the list of keys to look for while filtering a dictionary. It is first split on basis of semicolon(;) which acts like a separator of scenarios given. Next it’s split on comma which is the list of keys that the scenario mean.

roast.utils.roast_utils.get_flattened_dict(dictionary, remove_element=None)[source]¶

This api converts a given nested dictionary into a flattened one, by converting all the nested keys into combinations of tuple keys.

For Example: mydict = {} mydict[‘a’][‘b’] = 1 mydict[‘a’][‘b’][‘c’] = 2

output = get_flattened_dict(mydict)

output { (‘a’, ‘b’) = 1, (‘a’, ‘b’, ‘c’) = 2 }
